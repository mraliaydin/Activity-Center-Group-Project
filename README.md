# Activity-Center---Class-Group-Project

A simple Java project that demonstrates an Activity Center.
It is a group project for CTIS221 class.
<br>
<br>
Users can:
<br>
-Select activity classes, 
<br>
-Select instructors,
<br>
-See details about courses,
<br>
-Make registration.
